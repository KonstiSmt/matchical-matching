# Matchical Slides Style Guide (v1)

This file captures the currently provided slide style guidance for the code-based deck system.

## Brand Intent
- Minimalistic, low-clutter, modern presentation style.
- Strong contrast with Matchical orange/navy identity.
- Keep emotional cover and closing slides as strong anchors.
- Do not copy old slide layouts; use old deck as content inspiration only.

## Color Tokens
### Primary
- Main: `#FF5E00`
- Dark: `#C42800`
- Light: `#FF903E`
- Contrast: `#FFFFFF`

### Secondary
- Main: `#002839`
- Light: `#305063`
- Dark: `#000014`
- Contrast: `#FFFFFF`

### Text
- Primary: `#002839`
- Secondary: `#305063`
- Disabled: `#E9F2F6`

### Background
- Paper: `#FFFFFF`
- Default: `#F8F8F8`

### Status Colors
- Error Main: `#D32F2F`
- Error Dark: `#C62828`
- Error Light: `#EF5350`
- Warning Main: `#FFF73B`
- Warning Dark: `#CBC400`
- Warning Light: `#FFF771`
- Success Main: `#2E7D32`
- Success Dark: `#1B5E20`
- Success Light: `#4CAF50`
- Info Main: `#0288D1`
- Info Dark: `#01579B`
- Info Light: `#03A9F4`

## Typography Preferences
- Heading font: Steradian (as provided)
- Body/subtitle font: Commisioner (as provided)
- Use robust fallbacks until brand font files are integrated.

## Slide Direction Notes
- Cover direction reference: "We're matchical" style.
- Closing direction reference: "That's matchical" style.
- Content slides should support clean storytelling with generous spacing.
