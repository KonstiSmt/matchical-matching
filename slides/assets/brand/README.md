# Brand Assets

This folder stores versioned slide-brand assets for the code-based presentation system.

## Logos
- `logos/Matchical_Logo_Default.svg`
- `logos/Matchical_Logo_Inverse.svg`
- `logos/Matchical_Logo_Symbol.svg`

## Usage
- Use `Matchical_Logo_Default.svg` on light backgrounds.
- Use `Matchical_Logo_Inverse.svg` on dark backgrounds.
- Use `Matchical_Logo_Symbol.svg` for decorative accents, markers, and visual motifs.
