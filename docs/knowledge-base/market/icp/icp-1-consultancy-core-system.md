# ICP 1: Consultancy Core System

Consultancies that use Matchical as a core internal operating system for consultant data, demand matching, and submission output.

## Confirmed Facts
- Primary goal: manage internal consultant data and deploy consultants into project demands efficiently.
- This ICP is internal-first in focus.
- Key value areas:
  - Structured consultant profile management.
  - Opportunity and demand parsing support.
  - Internal search and matching workflows.
  - CV export across customer-required templates and languages.
- Consultant profile structure includes skills, roles, certificates, languages, education, training, and project or engagement context.
- External sourcing-heavy workflows are primarily ICP 2 and usually involve different buying centers and decision makers.
