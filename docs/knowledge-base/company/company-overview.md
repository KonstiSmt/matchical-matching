# Company Overview

Matchical is a B2B SaaS software provider for consultancies and external staffing agencies that staff high-skilled consulting talent into project engagements.

## Confirmed Facts
- Matchical provides software; Matchical itself is not doing staffing operations.
- Matchical is not a permanent hiring or ATS product.
- Matchical focuses on staffing and deployment workflows for consulting talent in project contexts.
- Current go-to-market focus is DACH-first.
- Matchical currently operates with three behavior-based ICP modes:
  - Consultancy Core System (ICP 1)
  - External Staffing Agency (ICP 2)
  - Framework Orchestrator (ICP 3)
- One customer organization can map to one, two, or all three ICP modes.
